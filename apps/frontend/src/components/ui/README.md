# UI Primitives

This directory is for small reusable UI building blocks that can be shared
across frontend surfaces.

- `virtual-list.tsx`
  Lightweight virtualized list used by library, history, object-tree, and
  result views.

Only place code here when it is reusable outside one FEM-specific workflow.
