How to use the desktop shelf

  • Use the mirrored Hub pages for short operator-facing reading inside the desktop shell.

  • Use Current line before changing product direction or release posture.
  • Use Operations, Install integrity, and Update visibility for desktop-runtime behavior.
  • Use Language packs when UI copy needs to grow into more locales without forking the built-in translation set.
  • Use Troubleshooting when something looks partial, unhealthy, or inconsistent with the documented contract.
  • Use Testing and CI when the issue is workflow layout, operator search, or workflow-builder regression.
