Retained pack

  • Heat quad precompute no longer clones the full node list for every split triangle.
  • Thermal quad benchmarking keeps stage RSS visibility so later tuning has a stable reference.
  • Derived sparse matrices use row-capacity hints and sorted append instead of repeated binary-search insertion.
  • The solve success path no longer keeps one extra long-lived scaled sparse matrix alive after compression.

Rejected ideas

  • Global unordered assembly plus one final row merge for the heat quad path.
  • Re-reading diagonals on every CG iteration.
  • Treating stage-end RSS as if it fully explained release-mode peak RSS.

Current benchmark anchor

  • Local optimized path: repeat 3, median 657.5775 ms, peak RSS 15 MiB.
  • Clean control group: repeat 3, median 6797.5630 ms, peak RSS 22 MiB.
  • Physical lab release: repeat 3, median 30.8742 ms, peak RSS 29 MiB.
