# Hub GUI Icons

These are the packaged desktop icons for the Tauri Hub shell.

- `kyuubiki-app.png`
  primary app icon used for the desktop shell bundle
- `kyuubiki-app.icns`
  macOS icon bundle source used by Tauri packaging
- `kyuubiki-app.ico`
  Windows icon bundle source used by Tauri packaging
