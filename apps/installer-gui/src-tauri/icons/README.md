# Installer Icons

This directory contains the packaged desktop icons for the Tauri installer.

- `kyuubiki-app.png`
  Main application icon used for the installer window/app bundle.
- `kyuubiki-app.icns`
  macOS application icon bundle.
- `kyuubiki-app.ico`
  Windows application icon bundle.
